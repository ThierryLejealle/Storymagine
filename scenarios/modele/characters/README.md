# Fiches personnages

Un fichier par personnage, nommé `prenom_nom.md`.

Chaque fiche comporte trois sections, routées automatiquement vers les bons agents :

## GÉNÉRAL *(obligatoire)*
Identité publique : âge, origine, profession, traits dominants.
Partagé entre le Planner et le Writer.

## INTRIGUE *(optionnel)*
Arc narratif, motivations profondes, vie privée, inconscient.
Injecté dans le Planner uniquement.
À montrer par l'action — jamais nommé explicitement dans le texte.

Les sous-sections `# privé` / `# inconscient` sont libres dans cette section.

## DESCRIPTION *(optionnel)*
Instructions d'écriture : voix, gestes, tics, manières.
Injecté dans le Writer uniquement.

La première ligne doit préciser l'article et le pronom :

```
Article : le — il
Article : la — elle
Article : l'Inspecteur — il
```

Les vérificateurs de cohérence ont accès aux trois sections.
