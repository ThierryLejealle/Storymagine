# Fiches personnages

Un fichier par personnage, nommé `prenom_nom.md`.

Chaque fiche comporte deux sections obligatoires et une section optionnelle :

## PLAN
Informations narratives (âge, rôle, arc, relations).
Injecté dans le Planner uniquement.

## RÉDACTION
Instructions d'écriture (voix, gestes, tics, manières).
La première ligne doit préciser l'article et le pronom :

```
Article : le — il
Article : la — elle
Article : l'Inspecteur — il
```

Injecté dans le Writer uniquement.

## INTÉRIEUR *(optionnel)*
Vie privée et inconscient du personnage — jamais nommé directement dans le texte,
manifesté uniquement par des gestes, silences ou réactions involontaires.
Injecté dans le Planner uniquement (pas le Writer).
